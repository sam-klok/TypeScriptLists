This demo of basic TypeScript operations with array

1. Install TypeScript locally:
npm install typescript

if you install it globally ( -g ), then add it to path

3. compile with local compiler with map for debugging:
.\node_modules\.bit\tsc -sourceMap

4. or compile and use settings file:
tsc --project tsconfig.json

5. run
node listDemo

5. you can run by F5 in VS Code,
then push button "Enter", and switch to Debug Console
